## 작업 내용

### ipynb 변환기 추가 (`ipynb_converter.py`)
- 코랩/Jupyter 노트북(.ipynb)을 마크다운으로 변환하여 NotebookLM 패키지에 포함
- 마크다운 셀 → 그대로, 코드 셀 → 코드 블록, 출력 → 텍스트만 (이미지 제외)
- 메타데이터(kernel, execution count) 제거
- pack 시 school_sync downloads에서 .ipynb 자동 감지 → 변환
- 이미 변환된 파일이 원본보다 새로우면 스킵 (불필요한 재변환 방지)

### 감사에서 수정한 문제
- **packer.py**: `materials_path` 구성이 `find_ipynb_files`와 불일치 → `_find_materials_dir` 헬퍼로 통일 (부분 매칭)
- **ipynb_converter.py**: `convert_and_save`에서 mtime 비교로 변경 없으면 스킵

### school_sync 중복 다운로드 수정 (`materials.py`)
- **원인**: `_try_download`에서 `_unique_path`가 같은 파일이 있으면 `파일_1.pdf`, `파일_2.pdf`로 이름 바꿔서 무한 저장
- **수정**: 파일이 이미 존재하고 크기가 MIN_DOWNLOAD_SIZE_BYTES 이상이면 다운로드 스킵, `skipped: true` 플래그 반환
- `_unique_path` 함수 제거 (데드 코드)
- skipped 파일은 cache에 mark_collected하지 않음 (원래 캐시가 관리)
