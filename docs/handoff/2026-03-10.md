### 이전 세션(03-09)에서 이어서 진행

#### 출력 구조 변경
- 날짜별 폴더(`과목_날짜/`) → **과목별 폴더(`과목/`)**로 전환
- 공유 파일(가이드, 컨텍스트, README)은 1개만, 전사본만 날짜별로 누적
- 같은 과목에 여러 날짜 SRT가 있으면 한 번에 전부 처리

#### sync_name 매핑 추가
- 다글로 폴더명과 school_sync 과목명이 다를 수 있는 경우 대비
- `config.yaml`에서 `courses.{과목}.sync_name`으로 매핑 가능
- `AppConfig.resolve_sync_name()` 메서드 추가

#### 컨텍스트 freshness 검증 + 재생성
- pack 시 학습 컨텍스트의 YAML frontmatter에서 `target_date`, `generated_at` 확인
- 전사본 날짜보다 오래됐으면 경고 + `context_export.py --course --date`로 재생성
- school_sync `context_export.py`에 `--course`, `--date` CLI 옵션 추가

### school_sync 변경사항
- `browser.py`: eClass 로그인 시 공지 팝업(.notice_popup) 자동 닫기
- `context_export.py`: `target_date` 파라미터 추가, CLI 지원, 과목별/날짜별 생성 가능
- `materials.py`: 다운로드 시 `manifest.json` 생성 (파일명/크기/다운로드 시각)
- 컨텍스트에 수업자료 파일 목록 섹션 추가
